# **第二部分：核心渗透技术专题**

## 专题一：无密码认证测试

### 2.1 技术介绍

无密码认证测试是指对不使用传统密码的认证机制进行全面的安全评估。其本质是**验证无密码认证系统能否安全地验证用户身份，防止令牌泄露、重放和账户接管**。

无密码认证常见类型：

| **类型** | **描述** | **安全风险** |
| :--- | :--- | :--- |
| 魔术链接 | 邮件中的一次性登录链接 | 链接泄露、劫持 |
| 邮件验证码 | 邮件发送 OTP | 邮箱接管 |
| 短信验证码 | 短信发送 OTP | SIM 交换、拦截 |
| WebAuthn | FIDO2 硬件/平台认证 | 钓鱼、重放 |
| 生物识别 | 指纹、面部识别 | 绕过、重放 |
| 社交登录 | Google/微信等第三方 | 第三方泄露 |
| 设备信任 | 信任设备自动登录 | 设备窃取 |

### 2.2 测试常见于哪些业务场景

| **业务场景** | **功能示例** | **风险点描述** |
| :--- | :--- | :--- |
| **魔术链接登录** | "发送登录链接到邮箱" | 链接泄露、有效期过长 |
| **短信登录** | "手机号 + 验证码登录" | SIM 交换、验证码爆破 |
| **邮件登录** | "邮箱验证码登录" | 邮箱接管、验证码泄露 |
| **生物识别登录** | TouchID/FaceID 登录 | 生物特征绕过 |
| **WebAuthn 登录** | 硬件密钥登录 | 钓鱼攻击、注册劫持 |
| **社交登录** | "通过 Google 登录" | OAuth 实现不当 |
| **设备信任** | "信任此设备" | 设备指纹伪造 |

### 2.3 漏洞探测方法

#### 2.3.1 黑盒测试

- **魔术链接测试**
  - 测试链接有效期
  - 测试链接一次性使用
  - 测试链接与用户绑定
  - 测试链接参数篡改
  - 测试多设备同时使用

- **验证码测试**
  - 测试验证码长度和复杂度
  - 测试速率限制
  - 测试验证码有效期
  - 测试重放保护
  - 测试用户绑定

- **WebAuthn 测试**
  - 测试注册流程安全
  - 测试认证流程安全
  - 测试钓鱼防护
  - 测试重放保护

- **生物识别测试**
  - 测试本地验证绕过
  - 测试重放攻击
  - 测试降级攻击

#### 2.3.2 白盒测试

- **代码审计**
  - 检查令牌生成逻辑
  - 审计验证逻辑
  - 检查速率限制实现
  - 查找硬编码绕过

- **配置检查**
  - 检查令牌有效期配置
  - 检查速率限制配置
  - 检查 WebAuthn 配置
  - 检查日志记录

### 2.4 漏洞利用方法

#### 2.4.1 基础信息收集

```bash
# 拦截魔术链接请求
# 分析链接格式、令牌结构、有效期

# 测试验证码枚举
for i in {1..100}; do
  curl -X POST https://target.com/verify \
    -d "email=user@example.com&code=000$i"
done

# 测试链接重放
# 使用已使用的链接再次尝试登录
```

#### 2.4.2 魔术链接攻击

| **攻击方法** | **描述** | **成功率** |
| :--- | :--- | :--- |
| 链接劫持 | 拦截未加密邮件 | 中 |
| 链接重放 | 重复使用链接 | 中 - 高 |
| 令牌预测 | 预测令牌生成 | 低 - 中 |
| 邮箱接管 | 接管用户邮箱 | 高 |
| 参数篡改 | 修改链接参数 | 中 |

#### 2.4.3 验证码攻击

```bash
# 暴力破解验证码
hydra -l user@example.com -P codes.txt \
  https://target.com http-post-form \
  "/verify:email=^USER^&code=^PASS^:Invalid"

# 测试速率限制
for i in {1..1000}; do
  curl -X POST https://target.com/send-code \
    -d "email=user@example.com"
done
```

#### 2.4.4 WebAuthn 攻击

- **钓鱼攻击** - 伪造 WebAuthn 请求
- **注册劫持** - 在注册时插入恶意凭证
- **重放攻击** - 重放认证响应
- **降级攻击** - 强制使用弱认证方式

### 2.5 漏洞利用绕过方法

#### 2.5.1 绕过速率限制

- **IP 轮换** - 使用代理池
- **邮箱变异** - 使用 `user+1@example.com` 等
- **时间分散** - 延长请求间隔
- **多通道** - 同时使用邮件和短信

#### 2.5.2 绕过设备信任

- **设备指纹伪造** - 修改设备标识
- **Cookie 复制** - 从已授权设备复制
- **User-Agent 伪造** - 伪装成已知设备

#### 2.5.3 绕过生物识别

- **Hook 验证函数** - 使用 Frida 等工具
- **重放响应** - 重放成功验证的响应
- **降级到密码** - 如果支持降级

# **第三部分：附录**

## 3.1 无密码认证测试检查清单

| **检查项** | **测试方法** | **预期结果** |
| :--- | :--- | :--- |
| 链接有效期 | 延迟使用链接 | 应拒绝过期链接 |
| 链接一次性 | 重复使用链接 | 应拒绝重用 |
| 用户绑定 | 交叉使用链接 | 应拒绝 |
| 验证码长度 | 分析验证码 | 应至少 6 位 |
| 速率限制 | 高频尝试 | 应限制 |
| 账户锁定 | 多次失败 | 应锁定 |
| 邮箱验证 | 修改邮箱参数 | 应验证所有权 |
| 钓鱼防护 | WebAuthn origin 检查 | 应验证 origin |

## 3.2 无密码认证测试工具

| **工具名称** | **用途** | **命令示例** |
| :--- | :--- | :--- |
| **Burp Suite** | 拦截/修改请求 | 手动测试 |
| **Hydra** | 暴力破解 | `hydra -l user -P codes.txt` |
| **Frida** | 动态插桩 | 生物识别绕过测试 |
| **WebAuthn Tester** | WebAuthn 测试 | 浏览器开发工具 |
| **自定义脚本** | 自动化测试 | Python/Go 编写 |

## 3.3 修复建议

1. **短期有效** - 魔术链接/验证码有效期不超过 15 分钟
2. **一次性使用** - 使用后立即失效
3. **速率限制** - 实施严格的发送和尝试限制
4. **用户绑定** - 令牌与用户会话严格绑定
5. **HTTPS 强制** - 所有无密码认证使用 HTTPS
6. **邮箱验证** - 验证邮箱所有权
7. **钓鱼防护** - WebAuthn 严格验证 origin
8. **审计日志** - 记录所有无密码认证事件
9. **备用方案** - 提供安全的备用认证方式
10. **用户通知** - 登录时通知用户
