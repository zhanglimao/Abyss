# **第二部分：核心渗透技术专题**

## 专题一：移动认证测试

### 2.1 技术介绍

移动认证测试是指对移动应用的身份验证机制进行全面的安全评估。其本质是**验证移动应用在本地存储、网络传输、会话管理等方面的认证安全性**。

移动认证特点：

| **特点** | **描述** | **安全风险** |
| :--- | :--- | :--- |
| 本地存储 | Keychain/Keystore、SharedPreferences | 敏感数据泄露 |
| 网络通信 | HTTPS、证书绑定 | 中间人攻击 |
| 生物识别 | TouchID、FaceID、指纹 | 绕过验证 |
| 设备绑定 | 设备指纹、Device ID | 设备模拟 |
| 离线认证 | 本地缓存凭证 | 凭证提取 |

### 2.2 测试常见于哪些业务场景

| **业务场景** | **功能示例** | **风险点描述** |
| :--- | :--- | :--- |
| **应用登录** | 用户名密码登录、生物识别 | 凭证存储不当、绕过 |
| **会话保持** | 记住登录状态、自动登录 | Token 泄露、长期有效 |
| **第三方登录** | 微信/QQ/Google 登录 | OAuth 实现不当 |
| **短信验证** | 短信验证码登录 | 验证码绕过、重放 |
| **设备绑定** | 新设备验证 | 设备指纹伪造 |
| **推送认证** | 推送通知确认登录 | 推送劫持 |
| **手势密码** | 图案解锁 | 图案泄露、暴力破解 |

### 2.3 漏洞探测方法

#### 2.3.1 黑盒测试

- **网络流量分析**
  - 使用代理工具拦截 HTTP/HTTPS 请求
  - 分析认证请求的参数和响应
  - 测试证书绑定绕过
  - 测试请求重放

- **本地存储分析**
  - 检查 iOS Keychain 存储内容
  - 检查 Android SharedPreferences/Keystore
  - 检查 SQLite 数据库中的敏感数据
  - 检查缓存文件中的凭证

- **应用逆向分析**
  - 反编译 APK/IPA 文件
  - 分析硬编码密钥和逻辑
  - 查找调试检测绕过方法
  - 分析生物识别实现

#### 2.3.2 白盒测试

- **代码审计**
  - 检查认证逻辑实现
  - 审计本地存储加密
  - 检查网络通信安全
  - 查找硬编码凭证

- **配置检查**
  - 检查 Info.plist/AndroidManifest.xml
  - 检查 ATS/网络安全配置
  - 检查调试配置
  - 检查混淆配置

### 2.4 漏洞利用方法

#### 2.4.1 基础信息收集

```bash
# Android 应用反编译
apktool decode app.apk
jadx -d output app.apk

# iOS 应用分析
unzip -q app.ipa
otool -l Payload/App.app/App

# 提取存储数据（需要 root/越狱）
adb shell run-as com.app cat shared_prefs/config.xml
```

#### 2.4.2 本地存储攻击

| **操作** | **方法** | **目的** |
| :--- | :--- | :--- |
| Keychain 提取 | 越狱设备使用 Keychain Dumper | 获取存储的凭证 |
| SharedPreferences 读取 | root 设备访问数据目录 | 获取 Token/配置 |
| 数据库提取 | 导出 SQLite 数据库 | 分析存储的数据 |
| 日志分析 | 读取应用日志 | 查找泄露的敏感信息 |

#### 2.4.3 网络攻击

```bash
# 使用 Frida 绕过证书绑定
frida -U -f com.app -l ssl-unpinning.js

# 使用 Mitmproxy 拦截流量
mitmproxy --mode transparent

# 重放认证请求
curl -X POST https://api.app.com/login \
  -H "Authorization: Bearer captured_token"
```

#### 2.4.4 生物识别绕过

- **Frida Hook** - Hook 生物识别验证函数
- **Xposed 模块** - 修改验证逻辑
- **内存修改** - 修改验证结果变量
- **重放攻击** - 重放验证成功的响应

### 2.5 漏洞利用绕过方法

#### 2.5.1 绕过证书绑定

- **Frida 脚本** - 使用开源 ssl-unpinning 脚本
- **Xposed 模块** - JustTrustMe 等模块
- **修改应用** - 反编译后移除证书验证代码
- **系统级绕过** - 使用定制 ROM

#### 2.5.2 绕过 Root/越狱检测

- **Magisk Hide** - 隐藏 Root 状态
- **Xposed 模块** - 绕过检测逻辑
- **Frida Hook** - Hook 检测函数返回 false
- **修改应用** - 移除检测代码

#### 2.5.3 绕过设备绑定

- **设备信息修改** - 使用工具修改 Device ID
- **应用克隆** - 使用平行空间等工具
- **模拟器** - 使用模拟器伪造设备信息
- **Token 提取** - 从已授权设备提取 Token

# **第三部分：附录**

## 3.1 移动认证测试检查清单

| **检查项** | **测试方法** | **预期结果** |
| :--- | :--- | :--- |
| 本地存储加密 | 检查存储的 Token | 应加密存储 |
| 证书绑定 | 尝试 MITM 拦截 | 应拒绝连接 |
| Root 检测 | 在 Root 设备上运行 | 应检测并拒绝 |
| 调试检测 | 附加调试器 | 应检测并退出 |
| 日志泄露 | 检查 Logcat/Console | 无敏感信息 |
| 生物识别 | 尝试 Hook 验证函数 | 应有防篡改机制 |
| 会话超时 | 长时间不操作 | 应自动注销 |
| 多设备登录 | 多设备同时登录 | 应有合理策略 |

## 3.2 移动认证测试工具

| **工具名称** | **用途** | **命令示例** |
| :--- | :--- | :--- |
| **Frida** | 动态插桩 | `frida -U -f app -l script.js` |
| **Objection** | 移动安全工具包 | `objection explore` |
| **MobSF** | 静态/动态分析 | Web 界面 |
| **JADX** | APK 反编译 | `jadx -d output app.apk` |
| **Burp Suite** | 流量拦截 | 配合代理设置 |
| **Xcode** | iOS 调试 | 调试 iOS 应用 |

## 3.3 修复建议

1. **安全存储** - 使用 Keychain/Keystore 存储敏感数据
2. **证书绑定** - 实施证书绑定防止 MITM
3. **代码混淆** - 使用混淆增加逆向难度
4. **完整性校验** - 检测应用是否被篡改
5. **Root 检测** - 检测并限制 Root 设备使用
6. **调试检测** - 检测调试器附加
7. **会话管理** - 实施合理的会话超时策略
8. **生物识别** - 结合后端验证，不依赖本地验证
9. **日志清理** - 生产环境禁用调试日志
10. **HTTPS 强制** - 所有通信使用 HTTPS
